
# Simulation Projects

## Overview

This project is a collection of interactive simulations built with Astro. Each simulation showcases different concepts and technologies.

## Implemented Simulations

*   **Forest Fire Simulator:** Experience the mesmerizing dance of a forest fire with this interactive simulator. Built with Svelte.
*   **Fish Boids:** A mesmerizing simulation of flocking behavior in fish. Built with Astro and JavaScript.
*   **City Generation:** A generative simulation of city growth and development. Built with React.
*   **Whirlpool Vector Field:** An interactive simulation of a whirlpool vector field. Built with Preact.
*   **Ant Colony Simulator:** An Ant Colony delving into the world of pheromones. Built with SolidJS and PIXI.js.

## Design

*   **Dark Theme:** The entire application uses a dark theme for a visually appealing experience.
*   **Card-based Navigation:** The home page features a grid of cards, each representing a different simulation.
